What is CSS Syntax?

CSS stands for Cascading Style Sheets, and is used to style the layout of web pages. It consists of rules that define how HTML elements should be displayed on screen, paper, or in other media.

Basic Syntax:

selector {
    property: value;
}

This is the basic syntax for writing CSS rules. A selector is an HTML element that you want to style, and the property and value define the style you want to apply.
